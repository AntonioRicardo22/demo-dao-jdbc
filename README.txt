CRUD de Vendedores e Departamentos em Java com JDBC

Este projeto é uma aplicação Java que implementa um CRUD completo para gerenciamento de vendedores e departamentos, utilizando JDBC puro com banco de dados MySQL.

🛠 Tecnologias utilizadas

- Java 17
- JDBC
- MySQL
- SQL
- Programação Orientada a Objetos

📦 Funcionalidades

- Cadastro, listagem, atualização e exclusão de vendedores (`Seller`)
- Cadastro e gerenciamento de departamentos (`Department`)
- Relacionamento muitos-para-um: cada vendedor pertence a um departamento

💾 Banco de Dados

O projeto utiliza MySQL. O script para criação do banco está na pasta `src/db/database.sql`.
